# 12-python
a repo containing 12 easy python codes/projects

## Projects

| # | File | Description |
|---|------|-------------|
| 1 | [01_hello_world.py](01_hello_world.py) | Greets the user by name |
| 2 | [02_calculator.py](02_calculator.py) | Basic calculator (add, subtract, multiply, divide) |
| 3 | [03_number_guessing.py](03_number_guessing.py) | Guess a random number between 1 and 100 |
| 4 | [04_temperature_converter.py](04_temperature_converter.py) | Convert between Celsius, Fahrenheit, and Kelvin |
| 5 | [05_todo_list.py](05_todo_list.py) | Simple interactive to-do list |
| 6 | [06_fibonacci.py](06_fibonacci.py) | Generate a Fibonacci sequence |
| 7 | [07_palindrome_checker.py](07_palindrome_checker.py) | Check if a word or phrase is a palindrome |
| 8 | [08_fizzbuzz.py](08_fizzbuzz.py) | Classic FizzBuzz up to a user-defined limit |
| 9 | [09_rock_paper_scissors.py](09_rock_paper_scissors.py) | Play Rock, Paper, Scissors against the computer |
| 10 | [10_password_generator.py](10_password_generator.py) | Generate a random password with custom options |
| 11 | [11_unit_converter.py](11_unit_converter.py) | Convert between km/miles and kg/pounds |
| 12 | [12_word_counter.py](12_word_counter.py) | Count words, characters, and sentences in text |

## How to Run

```bash
python <filename>.py
```
