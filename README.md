# 12-python

A repository containing 12 beginner-friendly Python mini projects.

## Project Aim

The aim of this project is to create a curated list of basic Python projects that help learners practice core programming concepts through small, practical scripts.

## What You Will Practice

- User input and output
- Conditional logic (`if/elif/else`)
- Loops (`for` and `while`)
- Basic functions and program structure
- String and number operations
- Randomization and simple game logic

## Prerequisites

- Python 3.8 or later
- A terminal (PowerShell, Command Prompt, or any shell)
- A code editor such as VS Code (optional, but recommended)

## Projects

| # | File | Description |

## How to Run

1. Open a terminal in the project folder.
2. Run any script using:

```bash
python <filename>.py
```

Example:

```bash
python 02_calculator.py
```

## Suggested Learning Path

If you are a complete beginner, try the projects in this order:

1. `01_hello_world.py`
2. `02_calculator.py`
3. `08_fizzbuzz.py`
4. `07_palindrome_checker.py`
5. `03_number_guessing.py`
6. Continue with the remaining files in numeric order

## Notes

- These projects are intentionally simple and focused on fundamentals.
- You can improve each script by adding validation, error handling, or extra features.
